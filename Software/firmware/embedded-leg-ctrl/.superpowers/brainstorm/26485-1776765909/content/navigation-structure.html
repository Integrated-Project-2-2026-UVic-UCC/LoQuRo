¿Cómo navegar la página?

Elige la estructura que mejor comunique el proyecto

A

Landing Page Scroll

Una sola página, todo fluye verticalmente. Hero → Demo → Arquitectura → Specs → Hardware → Equipo → Impacto. Ideal para mantener ritmo y momentum.

B

SPA con Navbar

Menú principal con secciones separadas (Home, Robot Overview, Architecture, Hardware, Team, etc.). Mejor para navegar rápido entre temas específicos.

C

Híbrido

Landing scrolleable pero con navbar sticky. Permite navegar rápido o explorar en profundidad según el usuario.
